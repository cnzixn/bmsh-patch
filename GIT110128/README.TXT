Instructions(使用方法)

	If necessary, please back up the unmodified official "original" files by yourself.
	如有需要，请自行备份未修改的官方“原版”文件。

	The application package (APK) and data package (OBB) are ZIP compressed files.
	“游戏”的安装包(APK)、数据包(OBB)为ZIP文件。

	Add all files in the folder ADD_TO_APK to the application package.
	进入ADD_TO_APK，将全部的文件“添加”到安装包。

	Add all files in the folder ADD_TO_OBB to the data package.
	进入ADD_TO_OBB，将全部的文件“添加”到数据包。

	After the application package is modified, it needs to be "signed" to be installed.
	安装包被修改后，需要“签名”才可以被正常安装。

	All "com" in the data package path needs to be changed to "hei".
	数据包路径中的2个“com”，都需要修改为“hei”。
